## Design Context

### Users
Tigerkid Yang and visitors reading a personal blog, project notes, and CS self-study Markdown. The site should feel exploratory, but reading long technical notes with code, images, gifs, and math must stay comfortable.

### Brand Personality
Playful, technical, unruly. The interface should feel like a personal terminal with attitude, not a generic dark developer portfolio.

### Aesthetic Direction
Retro terminal drama with CRT texture, strong green phosphor identity, amber and red system accents, visible command rhythm, and exaggerated interaction feedback. Keep the TigerKid ASCII mark as the memorable first-screen signal.

### Design Principles
- Preserve the terminal illusion while keeping navigation clickable and readable.
- Make the first screen feel alive without pushing the prompt to the bottom.
- Use bold effects as interface feedback, not random decoration.
- Keep technical articles calm enough for long-form reading.
- Respect reduced-motion preferences.
